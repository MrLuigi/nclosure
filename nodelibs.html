Node Core Lib Wrappers

NClosure also provides you with a set of wrappers that allow you to use the Node.js core libs as type safe Closure annotated classes.

To use the type-safe Node.js libs just: 

    require('nclosure').nclosure(); // Initialise the framework

    goog.provide('namespace');

    goog.require('goog.array');

    goog.require('node.fs'); // All node libs are in the "node" namepsace

    // Now just use the node utilties and benefit from type checking and
    // all other benefits of closure annotated code.
    console.log('Files in directory: ' + node.fs.readdirAsync('.'));
