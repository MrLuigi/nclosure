demo.listfiles(dir)
This constructor intiates the process of listing the details of all files in the specified directory. Note: This constructor is not meant to be used programatically. Rather it is ment to be executed using a node shell. Example: 
  ./listfiles
That will list all the files in the current directory.
Parameters:
{string} dir
The directory to list.
